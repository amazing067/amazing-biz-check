# 매니저 출석 체크

회사 TV와 태블릿으로 쓰는 간단한 출근/퇴근 체크 시스템입니다. 도메인 없이 같은 네트워크에서 PC/태블릿 IP로 접속해 사용합니다.

## 필요한 것

- **TV 화면**: 메인 대시보드 (오늘의 명언, 전체 매니저 연락처, 오늘 출근한 매니저)
- **태블릿 1**: 출근 체크 (매니저 선택 후 출석 체크)
- **태블릿 2**: 퇴근 체크 (퇴근 시 해당 매니저가 TV 화면에서 사라짐)

## 실행 방법

```bash
npm install
npm start
```

**개발 시 자동 새로고침** (파일 수정 시 브라우저 자동 갱신):

```bash
npm run dev
```

실행 후 브라우저에서 **`http://localhost:2000`** 으로 접속하면, `public/`·`data/` 파일을 수정할 때마다 자동으로 새로고침됩니다.

**도메인 없이** 개발 주소로 웹페이지 접속하면 됩니다.

- **이 PC에서**: 브라우저에 `http://localhost:2000` 입력 (또는 `npm start` 시 자동 오픈)
- **같은 LAN 다른 기기**(TV·태블릿): `http://[이 PC IP]:2000` 사용 가능

- **TV**: `http://localhost:2000/tv.html`
- **출근/퇴근 체크 (태블릿 하나)**: `http://localhost:2000/checkin.html` — 한 번 누르면 출근, 같은 사람 다시 누르면 퇴근

## 데이터 수정

- **매니저 목록**: `data/managers.json` 에서 회사/이름/연락처·로고 파일명 추가·수정
- **오늘의 명언**: `data/quotes.json` 에 문구 추가 (날짜별로 순서대로 노출)
- **회사 로고**: 기본 경로 `C:\Users\ok\cheongu-landing\assets\logos` 사용. 다른 경로는 실행 전에 환경변수 `LOGOS_PATH` 로 지정 (예: `set LOGOS_PATH=D:\logos` 후 `npm start`). 서버 실행 시 콘솔에 "로고 경로 사용" 또는 "로고 폴더 없음"이 출력됩니다.

## 주의

- 출근/퇴근 기록은 `data/attendance.json` 에 저장되며, 날짜가 바뀌면 다음 날부터 새로 쌓입니다.
- TV 화면은 약 4초마다 자동 새로고침됩니다.
